\doxysubsection{Detailed Description}
This is a singleton class which hold encapsulates the main renderwindow the player info, the instructions for navigation, resources. 

This class provides a convinient way to pass around the necessary information easily between the Screens of the \mbox{\hyperlink{class_game}{Game}}.
