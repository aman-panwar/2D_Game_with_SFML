\doxysubsection{Detailed Description}
Encapsulates the working of leaderboard. 

The class creates a gui object which shows the leaderboard The class is intended to be used as such\+:

create a object to use it as a gui element

use static function add\+Player(name of the player, score of the player) to add a new player to the list of player or to update the maxscore of an old user
